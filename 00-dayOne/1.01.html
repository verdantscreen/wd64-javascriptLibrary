Day One: Notes

Notes

  • When using Emmet, class is noted by a "." and id is noted by "#"
  • HTML stands for Hyper Text Markup Language
  • DOCTYPE states the type of our document. In this case, HTML.
  • Our HTML tag denotes our language to be in English per the lang="en"
  • The head tag houses metadata, or detailed data about our doc
  • UTF-8 specifies character encoding. Each character (letter, number, even spaces) has a numeric value.
  • Viewport notes the device we are reading our doc on
  • Title relects the name and displays on the tab within our browser
  • A style tag can help us inject our css into the doc (this is not best practice).
  • Links allow us to connect other files, such as our CSS (stylesheet)
  • The Body tag houses all other tags that we want to display onto our browser. This is what the user sees.
